Modulo - Usando o Repeat

Intro

O repeat é uma função usada no CSS Grid para repetir automaticamente um mesmo padrão de tamanho em várias colunas ou linhas. Ele evita que você escreva o mesmo valor várias vezes e deixa o código mais limpo e organizado.

Com o repeat, você diz quantas vezes algo deve se repetir e qual é o tamanho que será repetido.

Isso ajuda principalmente quando o layout possui muitas colunas ou linhas iguais.

O objetivo principal do repeat é simplificar a criação de grids, tornar o código mais fácil de manter e ajustar, e permitir que o layout seja alterado mais rapidamente.

Também existem formas automáticas do repeat, como auto-fill e auto-fit, que permitem que o navegador crie quantas colunas couberem na tela, deixando o grid mais responsivo sem precisar calcular nada manualmente.

Explicacao do codigo:

O codigo parar ser feito esse grid ficou da seguinte forma:

.cont-repeat {
display: grid;
grid-template-columns: repeat(3, 1fr); /* e o mesmo de 1fr 1fr 1fr */
grid-template-rows: repeat(1, 50px); /* o mesmo de 1 50px */
}

auto-fill

O auto-fill é um valor que pode ser usado dentro do repeat no CSS Grid. Ele faz com que o navegador crie automaticamente o maior número possível de colunas com o tamanho que você escolheu.

Isso significa que ele vai preencher toda a largura disponível da tela (ou do container) criando várias colunas iguais, sempre repetindo o tamanho definido. Ele continua criando novas colunas até não caber mais nenhuma dentro do espaço disponível. O resultado é um grid que se ajusta sozinho conforme o tamanho da tela do usuário, mantendo sempre colunas do mesmo tamanho.
