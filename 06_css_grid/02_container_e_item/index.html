Intro

Iremos comecar pelos dois maiores conceitos de CSS Grid que sao: container e item.

Como funciona:

Para fazermos qualquer tipo de estrutura grid, iremos ter que ter pelo menos duas divs ou elemento.

Como usar Grid

Parra comecarmos a usar o grid, iremos criar 7 divs, onde uma sera o container pai e as outras 6 dentro dela. Para usarmos o grid e simples, basta usarmos o CSS com a propriedade, displau: grid;, temos diversas formas de estar montando nossas colunas, algumas propriedades iremos definir no container pai e outras serao necessarias serem definidas no proprio item.

Regras no geral que se aplicam a todos os elementos podem ser usadas no container pai, e regras especificas para um determinado elemento, pode ser feito no item especifico.
