Modulo - Usando minmax

Intro

Usada basicamente para fazer um calculo para definir o valor minimo e maximo de alguma propriedade que iremos usar.

Podemos usar o minmax de varias formas diferentes, como exemplo temos uma propriedade que consegumos definir a altura padrao de todas as linhas, nao apenas de uma linha especifica como com grid-template-rows. Podemos usar a propriedade grid-auto-rows, com ela conseguimos definir um valor para todas as linhas sem termos problemas com o scroll de rolagem se fizessemos isso com o grid-template-rows.
