Aula 01 - Oque é Grid

Intro

CSS Grid é um sistema de layout bidimensional, ou seja, ele organiza elementos em linhas e colunas ao mesmo tempo, diferente do Flexbox, que trabalha melhor em uma direção (linha OU coluna).

Você usa Grid quando quer montar estruturas completas, como:

Sessões com várias colunas

  • Layouts de páginas
  • Galerias de imagens
  • Grids que se reorganizam sozinhos no responsivo

Quando usar Grid?

Use Grid quando:

  • Você quer controlar linhas e colunas ao mesmo tempo
  • Está criando um layout com estrutura
  • Precisa reorganizar elementos no responsivo sem mexer no HTML

Use Flexbox quando:

  • Você quer alinhar elementos em uma única direção
