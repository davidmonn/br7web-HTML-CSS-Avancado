Modulo - Unidade FR

Intro

A unidade FR foi criada justamente para uso com o grid, e representa uma fracao do espaco disponivel do grid. Ou seja quando usamos 1fr estamos usando uma unidade que e totalmente flexivel.
Esse tamanho tem uma variacao que vai de acordo com a width de nosso grid. Quando usamos por exemplo 2fr ele sera o dobro do 1fr.
