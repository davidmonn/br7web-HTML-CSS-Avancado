Explicacoes:

Podemos observar que alem de nossos itens, tambem foi adicionado um background-color em nosso cont principal para podermos entender melhor.

Podemos observar que podemos posicionar os elementos de nosso grid, dentro da area de nosso container. Dentro de nosso container pai que esta servindo como agrupador desse grid, pode ser usado atributos como justify-content: center; o ruim nesse cenario seria usar justify-content: space-around; pois quebraria nosso gap se tivermos usando. Podemos usar tambem align-itens: center;
