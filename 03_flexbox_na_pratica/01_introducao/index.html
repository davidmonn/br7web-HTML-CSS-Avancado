Flexbox - Introducao e Direcao

Intro flexbox

Flexbox e uma forma diferente de criarmos e organizarmos a estrutura de nosso layout/HTML.

Compartivo de com e sem Flexbox

Para criarmos um layout sem flexbox, normalmente criamos duas divs

Exemplo sem Flexbox:

Acima podemos observar dois quadrado, normalmente para deixarmos um ao lado do outro, usamos a propriedade de CSS float: left; mas se no caso, quisermos que ambos quadrados tenham tamanhos variaveis, teriamos que fazer alguamas contas e processos mais complicados. Ja com flexbox esse trabalho se torna muito mais simples, facil e intuitivo

Com flexbox

Flexbox basicamente significa uma caixa flexivel, ou seja, usar flexbox estamos basicamente dizendo que nossas divs, containers e etc serao mais adaptaveis.

Para ativarmos o flaxbox na div, usamos a propriedade: display:flex quando fazemos isso, dizemos que esse container tera um funcionamento flex

Padrao flexbox

Se nao especficarmos ao adiconarmos um display:flex, por padrao ele ira ser horizontal (linha) mas podemos mudar isso usando a propriedade flex-direction nela temos algumas opcoes:

  • row (linha):padrao
  • column (coluna): ira mudar para coluna
  • row-reverse (linha ao contrario): ou seja, com essa opcao comecara da direita para esquerda
  • column reverse: mesma logica do row-reverse.
