Imagens Responsivas: Propriedade Object-fit

Aqui veremos uma otima propriedade do CSS, para tratar sobre adaptabilidade de uma imagem, apos escolhermos uma imagem, podemos ir em nosso CSS e fazer as seguntes config:

CSS:

img {
width:400px;
height:400px;
object-fit: none, fill, cover, contain ou scale-down;
}

Alguns valores do object-fit:

  • fill (padrao:) Preenche todo o container, podendo distorcer a imagem para caber.
  • contain: Ajusta o conteúdo inteiro dentro do container, preservando as proporções. Pode deixar espaços vazios (letterbox).
  • cover: Cobre todo o container preservando as proporções. Pode cortar partes da imagem para preencher totalmente.
  • scale-down: Mostra o menor entre none e contain (ou seja, se a imagem já for menor que o container, não escala; se for maior, reduz para caber).
  • none: Não redimensiona. Mostra o conteúdo em seu tamanho original, podendo vazar para fora do container.
