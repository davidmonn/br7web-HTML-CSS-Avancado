Funcao var em media query

Podemos usar var, em media query, para podermos ter responsividade com a variavel, mas muitos cometem um erro muito frequente, onde criam uma variavel para ex um font-size padrao e na hora de fazer o media query, nao usa uma variavel. Ou melhor doque isso, podemos simplesmente criar uma var com escopo local, para ser usado apenas nessa parte de media query.

Vantagem:

Esse padrão melhora a manutenção do código: se você quiser ajustar o tamanho da fonte para telas grandes, só precisa mudar um único valor local, e todos os elementos que usam var(--font-size-padrao) vão se adaptar.
