CSS @media: como funciona (2/2)

Aqui veremos uma das coisas que mais serao usadas, pois com ela podemos fazer layouts com responsividade

Como funciona:

  • Você define uma condição (ex: largura da tela).
  • Dentro dessa condição, você escreve o CSS que quer aplicar somente quando essa condição for verdadeira.

Continuacao da aula anterior

Com o @media, podemos tambem controlar o tipo de media que esta aparecendo, ex: na parte de impressao, podemos colocar uma propriedade onde quando o usuario for fazer a impressao, ira remover ou adicionar algo da impressao.

Para fazermos isso podemos fazer da seguinte forma:

CSS:
@media only print e nele adicionamos as propriedades como:

@media only print {
h1 {
font-size: 40px;
font-weight: bold;
}


p {
font-size: 18px;
}

}

Ex CSS:

Se quisermos por exemplo remover o h1 podemos usar a a propriedade selecionando nosso h1 e usar display:none;

Obs: se no caso o codigo nao funcionar, usar apenas o @media print.
