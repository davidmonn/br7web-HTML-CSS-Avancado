Hack CSS - Altura Relativa a Largura

Quando adicionamos um vídeo ao nosso site, por exemplo do YouTube, ele não é responsivo por padrão. Isso significa que, ao diminuir o tamanho da tela (como em celulares), o vídeo pode ficar cortado ou desproporcional.

Para resolver isso, podemos aplicar um hack de CSS que permite manter a proporção do vídeo (como 16:9), ajustando automaticamente a altura com base na largura da tela.

Para isso, usamos dois containers:

  • O primeiro é o container externo, que define a proporção com um padding-top.
  • O segundo é uma div interna que conterá o vídeo (iframe) e ficará posicionada absolutamente dentro do container externo.

Com essa estrutura, conseguimos fazer com que o vídeo se encaixe corretamente em diferentes tamanhos de tela, mantendo sua proporção original de forma responsiva.
