Imagens Responsivas: picture e source

Anotação sobre picture, source e img:

Com a tag picture, podemos agrupar várias imagens para exibir a imagem mais adequada dependendo do tamanho da tela ou tipo de dispositivo.

A tag img dentro do picture funciona como uma imagem padrão (fallback). Ela será mostrada caso nenhuma das outras opções seja usada.

Podemos ter, por exemplo, 3 imagens diferentes:

  • Uma para celular (pequena)
  • Uma para tablet (média)
  • Uma para PC (grande)

Para definir qual imagem será exibida em cada situação, usamos as tags com suas propriedades (principalmente o atributo media).

O navegador escolhe automaticamente a melhor imagem para exibir, de acordo com as condições definidas (ex: largura da tela).

Assim, conseguimos fazer uma responsividade de imagens:

  • Melhorando a velocidade de carregamento
  • Garantindo uma boa aparência em diferentes dispositivos
