Funcao var

A função var() no CSS é muito útil no dia a dia, especialmente quando trabalhamos em sites com várias páginas. Ela permite criar variáveis CSS, onde podemos armazenar valores como cores, tamanhos de fonte, espaçamentos, entre outros.

Dessa forma, se usarmos a mesma cor ou o mesmo tamanho em vários lugares, basta criar uma variável para esse valor. Se um dia precisarmos mudar, não será necessário alterar em cada local manualmente — basta mudar o valor da variável e todas as páginas seguirão o novo padrão automaticamente.

Isso economiza tempo, facilita a manutenção e deixa o código mais organizado.
