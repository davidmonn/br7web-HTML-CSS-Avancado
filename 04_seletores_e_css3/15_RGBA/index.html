RGBA

Intro

Iremos fazer alguns teste com RGBA. Existe varias formas de especificarmos cores no CSS como ja mencionado anteriormente. Quando queremos escolher uma cor com transparencia, so temos uma alternativa sendo ela RGBA onde escolhemos 3 numeracoes, que irao dde 0 a 255.

RGB

  • Red
  • Green
  • Blue

RGBA

  • A (Alpha) - Taxa de transparencia da cor

O valor do A do RGBA, vai de 0 a 1 (Sendo 1 = 100%). Ex: se quisermos 50% sera 0.5 .

Muitos sites utilizam quando usamos para abrir uma foto por exemplo.
