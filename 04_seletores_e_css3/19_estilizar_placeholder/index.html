Estilizar placeholder

Intro

Aqui iremos ver como modificar um placeholder de um input, como por exemplo conseguir alterar a cor que fica o placeholder.

Para fazermos isso iremos precisar de um outro CSS, isso pois nao iremos estilizar o input mas um conteudo dentro do input, entao para isso iremos precisar de uma outra pagina CSS

Tem formas diferentes de selecionarmos um placeholder, dependendo do navegador usado.

Navegadores Modernos

input::placeholder {color: red;}

Obs: input::placeholder podemos tambem fazer selecao de um item especifico ex:

input[type=text]::placeholder {
color: blue;
}

Para webkit, blink e edge

Navegadores que usam webkit podem ser feitos da seguinte forma: input::-webkit-input-placeholder {color: red;}

Firefox da verao 4 ao 18

input:-moz-placeholder {color: red;}

Firefox 19+

input::-moz-placeholder {color: red;}

Internet Explorer versao 10 e 11

input:-ms-placeholder {color: red;}

Microsoft Edge

input::-ms-placehoder {color: red;}
