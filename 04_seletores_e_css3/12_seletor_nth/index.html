Seletor NTH

Intro

Aqui iremos ver como selecionarmos, itens especificos, independente dele estar em primeiro, no meio ou ultimo. Para isso usamos um seletor NTH, nela possui algumas variacoes.

Como funciona NTH:

Supondo que temos 5 divs, e queremos escolher apenas a segunda para ser modificada sem precisar dar uma class ou id para ela, podemos usar o NTH, a seguinte forma por exemplo:

Exemplo 1 - nth-child

div:nth-child(2){
background-color: green;
}
Com isso sera selecionado apenas a segunda div.

Exemplo 2 - nth:last-child

Tambem temos outras opcoes com o NTH, como selecionarmos contando de baixo para cima, como exemplo iremos selecionar a div 4:

div:nth-last-chiel (2){
background-color: yellow-green; }
Podemos observar que selecionou a div 4, comecando a contagem da div 5
