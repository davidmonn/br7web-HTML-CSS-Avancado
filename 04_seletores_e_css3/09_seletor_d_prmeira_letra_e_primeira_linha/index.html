04 Modulo: Seletores e CSS 3 - Aula: Seletor de primeira letra e primeira linha

Intro

Quando possuimos muitos paragrafos, podemos basicamente pegar a primeira letra de cada paragrafo e aplicar um tipo de efeito. Antigamente para fazer isso era selecionado a primeira letra e adicionar por exemplo ela dentro de uma tag span, com isso o efeito era dado ao span. Hoje em dia nao precisamos mas fazer dessa maneira, ja com CSS podemos selecionar a primeira linha de cada paragrafo e aplicar as modificacoes.

Como fazer:

Para podermos fazer isso podemos por exemplo escolhendo a tag < p >

e em nosso CSS ficara da seguinte forma:
p::first-letter {

}
