Webkit

Intro Webkit

O Webkit é uma engine de renderização, ou seja, uma tecnologia responsável por desenhar na tela o conteúdo HTML e CSS de uma página web. Quando ouvimos que "um comando só funciona com Webkit", isso significa que o comando depende dessa engine específica para funcionar corretamente.

Cada navegador utiliza sua própria engine para interpretar o HTML, CSS e JavaScript. Apesar de todos seguirem os padrões da web (como os definidos pelo W3C), existem diferenças na forma como essas engines se comportam. Isso pode impactar no visual ou funcionamento de um site em diferentes navegadores.

Exemplos de engines por navegador:

  • Safari: usa o Webkit
  • Google Chrome: usa o Blink (baseado no Webkit, mas mais avançado)
  • Firefox: usa o Gecko
  • Opera: também usa o Blink

Por que isso importa ?

Saber qual engine está por trás de um navegador ajuda a entender por que certos comandos CSS (como propriedades com prefixos -webkit-) funcionam em alguns navegadores e não em outros. Isso também é útil ao fazer ajustes de compatibilidade entre navegadores.
