Tag Video

Intro Tag Video:

Nesta aula, vamos conhecer uma das tags mais populares do HTML5: a tag < video >. Ela revolucionou o uso de vídeos na web, pois eliminou a necessidade de usar plugins como o Flash. Com ela, podemos inserir um player de vídeo diretamente em nosso site de forma simples e eficiente.

Como usamos ?

A < video > é uma tag de abertura e fechamento, o que significa que ela abriga dentro dela outras tags ou elementos.

Nela temos alguns atributos que devemos inserir: width e height

Dentro de nossa tag, tambem podemos inserir mais alguams coisas como, a tag source que servira para adicionarmos a fonte de nosso video, ou seja, o arquivo ou link.

< video >

< source src="link" type="tipo de video" >

< /video >

Se deixarmos apenas dessa forma, nao teremos nem uma opcao de controle de nosso video, aparecendo apenas o primeiro frame. Para poder funcionar corretamente, adicionaremos o atributo: controls

Atributos para tag video/source

width e height: definem a largura e altura do player.

controls: adiciona os controles básicos (play, pause, volume, etc).

autoplay: faz o vídeo iniciar automaticamente ao carregar a página.

Tipos de vídeo suportados pelo HTML5 (type):

  • Dentro de nosso type temos 3 tipo suportados.
    • mp4
      • Suportado por todos os principais navegadores.
    • WebM
    • OGG/OGV
