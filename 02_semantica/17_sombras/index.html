Sombras

Intro:

As sombras podem ser aplicadas tanto em elementos (como caixas, botões, imagens) quanto em textos, usando as propriedades box-shadow e text-shadow.

Aplicacao

Para aplicar uma sombra em um elemento, usamos a propriedade box-shadow. Ela precisa de quatro valores principais para funcionar corretamente:

Ordem dos valores para box-shadow

  • 1. Distância horizontal - desloca a sombra para a direita (valor positivo) ou para a esquerda (valor negativo).
  • 2. Distância vertical - desloca a sombra para baixo (valor positivo) ou para cima (valor negativo).
  • 3. Desfoque (blur) - define o quanto a sombra será suavizada. Quanto maior o valor, mais borrada ela será.
  • 4. Cor da sombra - pode ser uma cor sólida (black, #000) ou com transparência (rgba).
