Tag audio

Intro:

A tag < audio > foi introduzida no HTML5 com o objetivo de adicionarmos sons e músicas em nossos sites, sem depender de plugins externos como Flash.

A estrutura da tag < audio > é muito parecida com a da tag < video > . Ela é uma tag de abertura e fechamento, e dentro dela usamos uma ou mais tags < source > para definir o(s) arquivo(s) de áudio

Normalmente para nosso type com src="som.mp3"(mp3) sera usado o type="audio/mpeg" audio/mpeg.

Se tivermos outros tipos de type como um audio OGG, podemos coloca-lo como alternativo dentro de nosso source, como alternativo, caso nao funcione o padrao ira para outra opcao.

Dentro de nossa tag audio, tambem iremos precisar de atributos como controls/autoplay

Outros atributos tambem usados:

loop: ao finalizar video/audio, ira se repetir automatico.

muted: ira iniciar a reproducao video/audio com volume no 0

preload: Define se o áudio será carregado automaticamente junto com a página, util quando usuaios possuem uma internet mais lenta

Formatos suportados de audio:

mp3: mais compatível com navegadores.

OGG: formato aberto, usado como alternativa.

WAV: alta qualidade, porém arquivos maiores.

`

Dica:

Usar várias tags < source > com formatos diferentes ajuda a garantir que o som será reproduzido em todos os navegadores. O navegador vai tentar carregar o primeiro formato compatível.
