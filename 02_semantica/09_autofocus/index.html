autofocus

Oque e autofocus:

O atributo autofocus em um elemento de formulário (como < input >, < textarea >, < select >) faz com que aquele campo receba o foco automaticamente quando a página carrega. Ou seja, o cursor já fica piscando nesse campo, pronto para o usuário digitar.

Importante saber:

Só deve haver um único elemento com autofocus por página, pois o navegador foca só no primeiro que encontrar.

Funciona só quando a página carrega, não funciona depois via JavaScript (para isso, precisa usar .focus()).
