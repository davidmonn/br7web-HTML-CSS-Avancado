Introdução a Listas de Descrição

Aqui iremos ver um pouco mais sobre semântica. Embora muitas vezes não haja efeito visual, essas tags ajudam na acessibilidade, principalmente para pessoas que utilizam leitores de tela.

Quando temos uma lista de informações com título e descrição (texto, imagens, outras listas), podemos usar as tags semânticas apropriadas: dl, dt e dd.

dl (description list)

A tag dl representa a lista como um todo. Dentro dela, usamos dt para os títulos e dd para as descrições.

Resumo:

Basicamente, a tag <dl> funciona de forma semelhante à <ul>, pois agrupa um conjunto de itens. No entanto, em vez de usar apenas itens de lista, utilizamos <dt> para os títulos e <dd> para as descrições.
