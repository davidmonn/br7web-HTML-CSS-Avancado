Recomendacao de biblioteca CSS

Aqui iremos ver uma recomendacao de biblioteca muito famosa, podendo acessar ela pelo GitHub, acessada clicando aqui.

Antigamente, para garantir que uma página funcionasse da mesma forma em diferentes navegadores, utilizávamos algo chamado CSS Reset. Ele servia para "resetar" todas as propriedades padrão dos elementos HTML, evitando comportamentos diferentes entre navegadores. No entanto, o CSS Reset tradicional tinha suas limitações.

  • Fornece um estilo base consistente entre diferentes navegadores;
  • Corrige bugs e inconsistências comuns no CSS padrão dos navegadores;
  • Mantém os estilos úteis por padrão, ao invés de removê-los totalmente como o CSS Reset tradicional;
  • Facilita o desenvolvimento de layouts mais previsíveis e responsivos;
  • É leve e fácil de usar — basta incluir no início do seu CSS.
