Tags de input

Novos tipos de input

O HTML5 trouxe novos tipos de input, que facilitam o trabalho sem precisar de muito JavaScript:

Alguns dos inputs:

input search:

Campo de pesquisa com botão para limpar.

input email:

Verifica se o texto digitado é um e-mail válido.

input url:

Aceita apenas links válidos.

input tel

Campo para telefone.

input namber:

aceita números com setas para aumentar/diminuir.

input range:

barra deslizante (usa atributo min e max).

input date:

Campo com calendário.

input week:

Seleciona uma semana.

input time:

Seleciona um horário.

input datetime-local

Com ele conseguimos selecionar data e hora.

input color:

Abre um seletor de cor.
