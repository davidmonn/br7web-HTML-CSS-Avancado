meta tag social sharing

Intro meta tag social sharing:

As meta tags de social sharing são trechos de código HTML usados para controlar como uma página aparece quando é compartilhada em redes sociais como Facebook, Twitter, LinkedIn, entre outras. Elas informam aos bots dessas plataformas o título, descrição, imagem e URL da página.

Como funciona:

A página precisa estar publicada online. Usamos a ferramenta Facebook Sharing Debugger para visualizar como o Facebook interpreta os dados da URL e forçar uma atualização (Scrape Again).

Para ele funcionar corretamente com: imagem, titulo, descricao e etc. Precisamos usar as principais meta tag, para funcionar essas configuracoes especificas.

1. Iremos comecar indo em nosso HTML

Para fazer as modificacoes/adicoes iremos0 em nosso head com a tag , o primeiro meta iremos definir o link do arquivo ou site acessando, e dentro desse meta iremos usar os atributos: meta property="og:url(item que iremos descreve)" content="(url o site)"

2. Tipo do site

< meta property="og:type" content="article(mais usada), ou se fosse produto de venda, podemos usar um product" >

3. Titulo

< meta property="og:title" content="Titulo o nosso arquivo" >

4. Descricao:

< meta property="og:description" content="A descricao do nosso arquivo" >

5. Imagem:

< meta property="og:image" content="link ou arquivo da imagem" > Por padrao recomendado pelo facebook para a imagem e de 1200*630 e no max 8mb

Obs:

A ordem nao importa essas sao apenas as padroes, foi feito dessa maneira para visualizar e entender as opcoes padroes, e oque cada uma fara, contato que tenha essas opcoes, basicamente tudo funcionara. Outras redes sociais, como o LinkedIn, também utilizam as tags OG.

Após configurar:

Podemos entrar no site novamente em nosso sharing debug podemos clicar em Scrape Again, ele ira pegar novamente as informacoesdo link passado, mas agora com tudo certo.
