Atributo data
Os atributos data-* foram introduzidos no HTML5 para permitir que desenvolvedores adicionem informações personalizadas aos elementos HTML. Essas informações não afetam a aparência do elemento e não são visíveis para o usuário, mas podem ser acessadas e manipuladas facilmente com JavaScript.

Para que serve o data-*?

Ele é útil quando você precisa armazenar dados no HTML, como:

  • ID de usuário
  • Nome
  • Informações de produto (preço, cor, código)
  • Status ou configurações específicas

Esses dados podem ser lidos depois via JavaScript para tomar decisões, fazer cálculos, mostrar mensagens personalizadas, etc.

Resumo:

  • data-* é um atributo para guardar dados personalizados no HTML.
  • Os valores podem ser acessados facilmente com JavaScript via element.dataset.
  • Útil para armazenar informações que serão usadas no lado do cliente sem precisar usar variáveis ou requisições imediatamente
