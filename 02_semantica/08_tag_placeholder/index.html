Oque e placeholder ?

O atributo placeholder em HTML é usado dentro de elementos de formulário (como < input >, < textarea >) para exibir um texto temporário que orienta o usuário sobre o que deve ser preenchido naquele campo.

Para que serve ?

Serve para ajudar o usuário, dando um exemplo ou sugestão do tipo de conteúdo que deve ser inserido naquele campo. Esse texto aparece dentro do campo, mas desaparece assim que o usuário começa a digitar ou clica para escrever.
